## 如何更新

1. 次版本更新（如 1.1.0 -> 1.2.0）：
```bash
git pull
pip install -r requirements.txt --upgrade
python manage.py migrate
```

2. 主版本更新（如 1.0.0 -> 2.0.0）：
- 请参考 [迁移指南](./MIGRATION.md)
- 可能需要手动迁移数据
- 需要更新依赖关系
